# FAQ

## Is this a monitoring tool?

No.

Sentra does not focus on reporting system status.

It focuses on determining whether a system is safe to act on.

---

## Does this replace observability platforms?

No.

Sentra operates above existing tools and uses their signals.

---

## Why not just use alerts?

Alerts indicate events.

They do not determine whether a system has actually returned to a stable condition.

---

## What makes this different?

Sentra introduces:

- explicit system states  
- recovery validation over time  
- action gating based on confidence  

---

## Is this open source?

Not at this time.

This repository provides a high-level view of the system.
